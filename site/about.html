À propos

Ce site est réalisé pour pour faire une démonstration de concept d’une application web pour la reconnaissance faciale. Dans cet espace, il sera possible de reconnaitre les personnes qui font partie de la base de données. Il sera même possible de modifier cette base de données par l’administrateur en ajoutant des personnes autorisées, d’en supprimer ou encore de modifier leurs images.
L’usage pourra être utilisé par n’importe qui, et n’importe quand tant qu’une connexion à internet est disponible. On pourra même ajouter un dispositif connecté à cette application pour diversifier l’usage comme par exemple un Arduino pour contrôler l’ouverture ou la fermeture d’une porte.
